# FarmTrack — Transparent Farm-to-Home Delivery App

## Overview

FarmTrack is a mobile application designed to bring transparency and trust to farm-to-home delivery services. The app allows customers to track their orders in real time through a clear status timeline, reducing confusion and improving customer satisfaction.

This project is developed using Flutter for mobile app development and Firebase for backend services.

---

## Problem Statement

Local farm delivery services often rely on manual coordination methods such as phone calls or messaging apps. Customers frequently lack visibility into order status and delivery timelines, which creates uncertainty and reduces trust.

FarmTrack solves this problem by providing a structured and trackable digital delivery journey from order placement to final delivery.

---

## Solution

The application provides a mobile-first platform where:

* Customers can monitor order progress
* Admins can update order status
* Data updates occur in real time using cloud backend services

## App Workflow

1. User registers or logs into the app
2. User views dashboard showing active orders
3. Each order displays a status timeline:

   * Order Placed
   * Packed
   * Dispatched
   * Delivered
4. Admin updates order status from backend panel
5. App reflects updates instantly using Firebase sync
