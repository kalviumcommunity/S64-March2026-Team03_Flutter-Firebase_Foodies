---

## What Are Custom Widgets  

In Flutter, everything is a widget. Custom widgets allow developers to create reusable UI components instead of rewriting the same code multiple times.

### Benefits  
- Reduces code duplication  
- Improves readability and structure  
- Ensures consistent UI design  
- Makes updates easier across the application  

---

## Types of Custom Widgets  

### Stateless Custom Widgets  
Used when the UI does not change after rendering.

**Use Cases:**  
- Cards  
- Labels  
- Static UI components  

### Stateful Custom Widgets  
Used when the UI changes dynamically based on user interaction.

**Use Cases:**  
- Buttons with state  
- Toggles  
- Interactive components
